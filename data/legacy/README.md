# Legacy Benchmark Data

This directory holds large benchmark CSV files generated by SageMath MILP solver
for ER, BA, and Regular graphs with N ∈ [4, 150].  These files are **not tracked
in the repository** (total uncompressed size ≈ 515 MB); they are archived on
Zenodo.

## Download

```bash
# Install zenodo-get (optional helper)
pip install zenodo-get

# Download all three files to this directory
# (replace ZENODO_DOI with the actual record DOI once published)
zenodo_get ZENODO_DOI --output-dir data/legacy/
```

Alternatively download manually from the Zenodo record and place the files here.

## Usage

These files are consumed by `experiments/regen_fig6_extended.py` to regenerate
Figure 6 (extended to N ≤ 150) with exact MILP reference covers.  The script
output (`results/results_dg_extended.csv`) **is** tracked and sufficient for
reproducing all paper figures and tables without re-running the full pipeline.

## Column schema

All three CSVs share the same columns:

| Column | Description |
|--------|-------------|
| `num_nodes` | Graph size N |
| `num_edges` | Number of edges |
| `graph_data` | Edge list as string: `"[(u, v, w), ...]"` |
| `graph_iteration_index` | Instance index (0–9 per setting) |
| `quantum_vc_size` | CTQW cover size |
| `exact_vc_size` | MILP optimal cover size |
| `2approx_vc_size` | 2-Approximation cover size |
| `fastvc_vc_size` | FastVC cover size |
| `sa_vc_size` | Simulated Annealing cover size |
| `quantum_approx_ratio` | `quantum_vc_size / exact_vc_size` |
| `graph_params` | JSON dict with generation parameters |
